# Проект футбол

## Проект предназначен для отображения и управления списком игроков футбольных команд.

## Запуск
Перед запуском убедитесь, что у вас установлены Docker и Docker Compose.

1. В корневом каталоге переименуйте файл: .env.example → .env
2. Запустите контейнеры: docker-compose up -d
3. После завершения запуска откройте в браузере [http://localhost:3000](http://localhost:3000)

## Использование
На странице будут отображаться текущие игроки и кнопки для добавления новых. Можете редактировать, удалять или добавлять игроков через интерфейс.
